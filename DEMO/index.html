1: data-accept-event

data-accept-event: will allow you to tell which type of event you accept to listen on the element
    possibles values can be any events (one ex: 'click' or several ex: 'click mouseover mouseout'),
    'all' (will accept all type of event), or empty '' (will deny all type of event).
    If not set then will be equal to 'all'.

2: data-target-type

data-target-type: will allow you to tell which type of target event can be sent from
    possibles values can be any those 5:
    'document', 'tag', 'class', 'id' which are the different type of event source we can catch,
    of course it is possible to set several of them
        ex: data-target-type='class id'
    'any' is the same as data-target-type='document tag class id'
    If not set then will be equal to 'any'.
